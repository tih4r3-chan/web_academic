Sobre App Academic

La creación de esta aplicación fue impulsada por la potencia de Ionic, un sólido Framework que proporciona una experiencia de desarrollo eficiente y una interfaz de usuario atractiva. La base de datos subyacente, alimentando la aplicación con agilidad y escalabilidad, se construyó sobre Firebase, específicamente utilizando Firestore.
En el aspecto visual, nos sumergimos en las posibilidades de diseño utilizando tanto SCSS como CSS para moldear el front-end. Esto nos permitió no solo cumplir con estándares de estilo modernos, sino también personalizar la experiencia del usuario de manera única.
Explore la sinergia entre la tecnología híbrida de Ionic y la flexibilidad de Firestore para crear una aplicación robusta y ágil. La combinación de estos elementos proporciona una experiencia fluida para los usuarios finales, respaldada por una infraestructura de backend potente y eficiente.
Estoy emocionada de presentar esta aplicación como resultado de la fusión de tecnologías avanzadas y enfoques innovadores para ofrecer una solución completa y moderna a las necesidades de nuestros usuarios.





La aplicación tiene como objetivo principal facilitar tanto a docentes como a alumnos el proceso de toma de asistencia y acceso a información. Nos enfocamos en hacer que estas tareas sean más simples y accesibles, priorizando la satisfacción de los usuarios que utilicen la aplicación. Queremos asegurarnos de que la experiencia sea cómoda y eficiente para todos los usuarios.
